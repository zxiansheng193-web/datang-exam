# 小程序图标说明

需要在 `miniprogram/images/` 目录下放置以下图标文件（建议尺寸：81x81 px）：

1. `exam.png` - 考试图标（未选中状态）
2. `exam-active.png` - 考试图标（选中状态）
3. `admin.png` - 管理图标（未选中状态）
4. `admin-active.png` - 管理图标（选中状态）

## 图标设计建议

### 考试图标
- 可以使用书本、笔、或者试卷的图标
- 未选中状态：灰色 (#999)
- 选中状态：蓝色 (#1e3c72)

### 管理图标
- 可以使用图表、设置、或者列表的图标
- 未选中状态：灰色 (#999)
- 选中状态：蓝色 (#1e3c72)

## 临时解决方案

如果暂时没有图标,可以从以下方式获取：
1. 使用 Iconfont 等图标库下载
2. 使用在线图标生成工具
3. 暂时注释掉 tabBar 配置中的 iconPath

## 示例图标下载

可以从以下网站免费获取图标：
- Iconfont: https://www.iconfont.cn/
- Flaticon: https://www.flaticon.com/
- IconFinder: https://www.iconfinder.com/
