## 🔧 高级使用

### 修改题库

如果您需要修改题库，用文本编辑器打开 `exam.html`，找到 `questionBank` 对象：

```javascript
const questionBank = {
    '前台接待顾问': {
        choices: [
            {
                question: '问题文本',
                options: ['选项A', '选项B', '选项C', '选项D'],
                answer: 0,  // 正确答案索引（0=A, 1=B, 2=C, 3=D）
                score: 5   // 分值
            }
        ],
        shortAnswers: [
            {
                question: '简答题问题',
                score: 10,
                answer: '参考答案'
            }
        ],
        scenarios: [
            {
                question: '情景题问题',
                score: 15,
                answer: '参考答案'
            }
        ]
    }
};
```

### 修改考试时间

找到 `remainingTime = 60 * 60;` 修改数字：
- 60分钟：`60 * 60`
- 90分钟：`90 * 60`
- 120分钟：`120 * 60`

### 修改标题

找到页面标题部分修改：
```html
<title>大唐环宇名车岗位技能考试系统</title>
<h1>大唐环宇名车岗位技能考试系统</h1>
```
